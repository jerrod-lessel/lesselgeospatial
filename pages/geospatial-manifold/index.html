The problem

Most urban resilience tools are built for emergency managers after disaster strikes. By then the road is already underwater, the bridge is already gone, and the evacuation route is already blocked. Geospatial Manifold is built for the people who can actually do something about it before that happens -- city planners, urban analysts, and resilience researchers who need to understand where their road networks are most vulnerable before the next earthquake, flood, or wildfire tests them.

How it works
Road network analysis

Network vulnerability mapping

Uses OSMnx to pull real road network data for any city or region, then runs graph-theory analysis via NetworkX to identify critical nodes and edges -- the roads and intersections whose failure would most severely fragment the network.

OSMnx NetworkX OpenStreetMap
Resilience scoring

Pre-disaster auditing

Generates resilience scores and vulnerability maps that planners can use to prioritize infrastructure investment, identify redundancy gaps, and model how the network would perform under various failure scenarios.
